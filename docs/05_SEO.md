## Metadata Rules

- Include exact dates only where relevant.
- Include "unofficial" or "checklist" in product-adjacent pages.
- Do not imply official status.
- Do not create duplicate thin pages for similar date queries.

## Internal Links

Every technical guide should link to:

- deadline page;
- checklist page;
- command decision page;
- relevant official source block;
- Gumroad/skill pack page.
