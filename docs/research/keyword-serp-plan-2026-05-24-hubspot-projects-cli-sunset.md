# Keyword and SERP Plan: HubSpot Projects CLI Sunset Kit

Run time: 2026-05-24 11:55 Asia/Shanghai

Scope: 20 keyword targets for a static SEO site plus Gumroad PDF/Notion/Sheet product. This is based on opened official docs, community threads, and current web search results available in this run. `super-search.web_search` and `super-search.fetch_url` were not available, so available web search/open tooling was used instead.

## SERP Summary

The SERP pattern is favorable for a checklist product:

- Official HubSpot docs rank or are the natural authoritative result for most core terms.
- Search results are fragmented across changelog, migration overview, new-app creation docs, CLI references, Marketplace listing docs, and legacy CRM card docs.
- Community pages reveal actual developer confusion, but are not packaged as a repeatable workflow.
- I did not find a clearly competing Gumroad-style paid "Projects CLI sunset checklist" in the sources reviewed this run.

Strategic implication: build pages that summarize, sequence, and source-link official documentation while selling a compact operational checklist. Do not try to outrank official docs on authority; win on long-tail task completion and deadline clarity.
