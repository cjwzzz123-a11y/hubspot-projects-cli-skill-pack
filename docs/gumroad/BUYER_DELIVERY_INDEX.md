## First 15-Minute Workflow

1. Open `projects-cli-preflight-checklist-2026-05-24.md`.
2. Classify the app: new public/Marketplace app, existing legacy public app, existing project-based app, existing private app, or CRM card modernization.
3. Record `hs --version`, Node version, target developer account, `hsproject.json` status, `platformVersion`, and `app-hsmeta.json` status.
4. Open the relevant official HubSpot source links before recommending a command.
5. Use `hubspot-developer-handoff-checklist-2026-05-24.md` to capture the final path to verify, open questions, owners, and source-check date.
