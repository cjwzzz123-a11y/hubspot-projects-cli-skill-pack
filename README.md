# HubSpot Projects CLI Skill Pack

Unofficial, source-linked launch project for a HubSpot Projects CLI migration skill pack, content site, and Gumroad digital product.

This repository is not affiliated with, endorsed by, or sponsored by HubSpot. HubSpot is used descriptively to identify the developer platform and public documentation this independent educational product references.

## Project Shape

- Static overseas SEO site for high-intent HubSpot Projects CLI and migration queries.
- Agent-ready skill pack for Codex/Cursor/Claude Code style workflows.
- Gumroad product artifacts: command cheatsheet, checklist, tracker, handoff template, source map.
- Official-source archive and source map for traceable product claims.

## Critical Claim Boundary

The May 26 / June 23, 2026 HubSpot event disables creation of new legacy public apps through the legacy Developer Platform UI. Existing legacy public apps continue functioning under that specific sunset. Do not imply an immediate shutdown of existing legacy public apps.
